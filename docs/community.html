ou Quartz - Support

User & Developer Community

Quartz is an open-source project, and is always on the lookout for feature ideas, enhancements, bug reports and all-around help. Find out how you can contribute (or get help) below.

User Forums

If you can't figure something out, just ask in our helpful forum. Quartz contributors and seasoned users are known to hang out here frequently!

Before posting a question, please double-check documentation (including JavaDOCs, the tutorial, and the docs in the distribution's "docs" directory). When you do post in the forum, please note the version of Quartz that you are using, information about the environment in which you are using Quartz (app server version, database version, etc.), and any logged information about the problem (full stack traces, etc.).

Bug Reporting / Feature Requests / Patch Submissions

Quartz uses the most excellent JIRA software to track issues.
(We might be biased though... JIRA uses Quartz as its scheduling component).

Use JIRA to report or submit the following:

  • Bugs - please be fairly certain the issue is, in fact, a bug.
  • Feature/Improvement requests
  • Code Contributions (patches or new features)

Paid (Commercial) Support

Commercial-quality support and training for Quartz is also available through one of the following channels:

  • From PartNET, Inc. based on hourly rates or contracted arrangements.
  • From James House (the primary developer of Quartz) based on negotiated pricing.

Donations

Although Quartz is free, your monetary donations are gladly accepted - they go directly to enabling the development and support of future versions of Quartz. Donations are accepted via the donation system on SourceForge.

Donations of code/development time/user support/etc. are all welcome as well. Please see the "Getting Involved" section below.

Getting Involved

Quartz development is hosted on SourceForge. Anyone is welcome to contribute in any way they wish - code, services, advice/suggestions, answering forum questions, etc.

Quartz 2.0 development is now underway! join the new Quartz 2 development mail list "dev@quartz.dev.java.net" to participate.

Also hosted on SourceForge, the quartz-developer mailing list is your ticket to getting involved with Quartz development. Join to get the inside scoop!

Anyone may access to the latest source code via CVS. You may browse the CVS tree or access it on Source Forge with via the project's cvs instruction page.
